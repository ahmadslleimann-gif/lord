# كيف تضيف منتج جديد عالموقع 🦁

## رفع الموقع عالهوستينغ 🚀

ارفع **كل محتويات** مجلد `lord-website` على `public_html` (من cPanel → File Manager أو FTP):
- `index.html` و `products.html` و `brands.html` و `admin.html`
- مجلد `assets` كامل (فيه الصور والـCSS والـJS)
- مجلد `api` (فيه ملف النشر `save-products.php`)

> ⚠️ **مهم**: إذا الموقع القديم لسا عالسيرفر، في ملف اسمو `index.php` — السيرفر بيفضلو على `index.html`. خود نسخة احتياطية من الموقع القديم وبعدين احذف أو غيّر اسم `index.php` حتى يفتح الموقع الجديد.

### كيف بيضيفو الأدمن منتجات بعد الرفع؟
أسهل ما يكون — **من المتصفح مباشرة، من أي جهاز**:
1. افتح `lordaccess.com/admin.html`
2. كلمة المرور: `LORD2026`
3. ضيف / عدّل / احذف متل ما بدك
4. اكبس **«نشر عالموقع ⚡»** واكتب كلمة المرور — **وخلص!** التعديلات بتصير ظاهرة فوراً لكل الزوار

الزر هاد بيظهر لحالو لما يكون الموقع عالهوستينغ (لأن الهوستينغ تبعك بيدعم PHP). وكل نشرة بتاخد نسخة احتياطية تلقائية بملف `products-data.backup.js` — إذا صار غلط، بتقدر ترجع عنها.

### تغيير كلمة المرور
لازم تغيرها **بملفين** (نفس الكلمة بالاتنين):
1. `assets/js/admin.js` — أول سطور: `var PASSWORD = 'LORD2026';`
2. `api/save-products.php` — أول سطور: `define('LORD_ADMIN_PASSWORD', 'LORD2026');`

## الطريقة السهلة: لوحة التحكم ⭐

افتح **admin.html** بالمتصفح:
- كلمة المرور: `LORD2026` (بتقدر تغيرها بأول ملف `assets/js/admin.js`)
- زر **«+ منتج جديد»**: بتعبي الاسم والسعر وبتختار صورة من جهازك — بتنضغط لحالها
- بتقدر **تعدّل** أو **تحذف** أي منتج من الجدول
- تعديلاتك بتنحفظ كمسودة بالمتصفح، ولما تخلص اكبس:
  - **«حفظ مباشر ⚡»** (كروم/إيدج): اختار ملف `assets/js/products-data.js` أول مرة — وبيحفظ عليه فوراً
  - أو **«نشر — تنزيل الملف»**: بينزل `products-data.js` جديد — استبدل فيه الملف القديم بمجلد `assets/js/`

> ⚠️ كلمة المرور حماية خفيفة بس (الموقع ثابت بدون سيرفر). إذا رح ترفع الموقع عالإنترنت، الأفضل ما ترفع `admin.html` معو أو تحميه من السيرفر.

## الطريقة اليدوية (اختياري)

```
assets/js/products-data.js
```

افتحو بأي محرر نصوص (Notepad بيكفي).

## الخطوات

### 1. حط صورة المنتج
انسخ صورة المنتج لمجلد:
```
assets/images/products/
```
مثلاً: `my-new-perfume.jpg`
(الأفضل صورة مربعة، 600×600 أو أكبر)

> 💡 إذا الصورة كبيرة (أكتر من 500KB)، شغّل `python _optimize_images.py` من مجلد الموقع — بيضغط كل الصور تلقائياً بدون ما يخرب جودتها.

### 2. ضيف المنتج بأول القائمة
افتح `assets/js/products-data.js` ورح تلاقي قائمة منتجات بهالشكل.
انسخ بلوك واحد، عدّل قيمو، وحطو **بعد أول قوس `[` مباشرة** حتى يطلع بأول الصفحة:

```js
{
  "id": 273,
  "name": "Dior Sauvage Elixir",
  "brand": "Dior",
  "price": 89.99,
  "oldPrice": 120.00,
  "image": "assets/images/products/my-new-perfume.jpg",
  "gender": "men",
  "category": "perfume",
  "description": "A rich, spicy elixir with lavender and woods."
},
```

### 3. معاني الحقول

| الحقل | القيم المسموحة |
|---|---|
| `id` | رقم جديد ما انعطى من قبل (زيد 1 عن أعلى رقم) |
| `name` | اسم المنتج |
| `brand` | اسم الماركة (إذا جديدة، بتظهر تلقائياً بصفحة الماركات*) |
| `price` | السعر الحالي بالدولار |
| `oldPrice` | السعر القديم إذا في تخفيض — أو `null` إذا ما في |
| `image` | مسار الصورة |
| `gender` | `"men"` أو `"women"` أو `"unisex"` |
| `category` | `"perfume"` أو `"eyewear"` أو `"bags"` أو `"accessories"` |
| `description` | وصف قصير (بيظهر بنافذة Quick View) |

> *ملاحظة: حتى تظهر ماركة جديدة بصفحة الماركات، ضيفها كمان بقائمة `LORD_BRANDS` بآخر نفس الملف (انسخ بلوك ماركة وعدّلو، و`count` هو عدد المنتجات).

### 4. احفظ الملف وخلص ✦
ما في داتابيز ولا كود — مجرد حفظ الملف والمنتج بيظهر فوراً مع كل الفلاتر والبحث والترتيب.

## أشياء انتبه عليها
- لا تنسى الفاصلة `,` بين منتج ومنتج
- الأسعار أرقام بدون `$` (مثلاً `42.99` مش `"$42.99"`)
- النصوص دايماً بين علامتي تنصيص `"..."`
- إذا الصفحة طلعت فاضية بعد التعديل → في غلطة كتابة بالملف (غالباً فاصلة ناقصة أو زيادة)
